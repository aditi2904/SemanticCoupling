1) The java classes were loaded from the path provided in the "args[0]". The text was scanned and Preprocessing was done on the scanned text, moreover the tfs of each java class was also calculated.
2) The inverted index and Idfs of all the java classes was calculated.
3) The Tf.Idfs was calculated of all the java classes was calculated.
4) The Cosine similarity of the target java class was calculated with all the other java classes.
5) The java classes were ranked according to cosine similarity.
6) The ranked java classes were printed.
7) The inverted index and Idfs were saved in a file on the disk.
8) The tf and Tf.Idfs of the target java class was saved in a file on a disk.
